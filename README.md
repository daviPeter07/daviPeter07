## Sobre mim

Sou um desenvolvedor de software dedicado à construção de aplicações eficientes, acessíveis e preparadas para evoluir.

Comecei minha jornada com PHP em 2024, desenvolvendo projetos pessoais que resolviam problemas reais. Essa base foi essencial para fortalecer minha lógica de programação, disciplina de código e visão prática sobre desenvolvimento de sistemas.

Ao longo da minha trajetória, tive contato com projetos em diferentes contextos, incluindo sistemas de gerenciamento, landing pages multilíngues, aplicações SaaS, dashboards, APIs REST, autenticação, controle de acesso, integrações com IA, filas, workers e publicação de aplicações.

Também possuo experiência com code review e responsabilidades de liderança técnica no front-end, apoiando decisões técnicas, definindo padrões, organizando demandas e contribuindo para a qualidade das entregas do time.

Hoje, além dos estudos e da evolução técnica constante, também lidero projetos digitais na SyncForge, atuando desde a estruturação técnica até a entrega final.

Gosto de trabalhar com código limpo, boas práticas, componentização, acessibilidade, SEO, testes, separação de responsabilidades e decisões técnicas alinhadas ao valor que o software precisa entregar.
